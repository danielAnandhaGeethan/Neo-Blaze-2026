# **App Name**: NeoBlaze 2026

## Core Features:

- Hero Section with Countdown: Display a captivating hero section with key event details and a real-time countdown timer.
- About Section: Present a concise and engaging overview of NeoBlaze, highlighting its purpose and anticipated impact.
- Themed Content Block: Create a visually distinct section dedicated to the event's theme, 'Teshuvah,' complete with relevant verses.
- Events Overview Grid: Showcase a grid-based display of all event categories with punchy taglines for each.
- Detailed Event Rules Accordions: Provide comprehensive event rules and guidelines within an expandable accordion format.
- General Rules Block: Always visible rules to avoid confusions. Registration fees, contact number...
- Scroll-Based Navigation: Enable smooth scrolling to different sections upon clicking top navigation links.

## Style Guidelines:

- Background color: Light beige (#F5F5DC) to create a warm and inviting atmosphere.
- Primary color: Burnt orange (#CC6633) for headings and highlights, reflecting a flame vibe.
- Secondary color: Deep charcoal (#333333) for main text, ensuring high readability.
- Accent color: Soft muted blue (#778899) as a secondary accent for buttons and subtle highlights.
- Headline font: 'Playfair' a modern sans-serif with an elegant, high-end feel; for titles such as 'NeoBlaze 2026' and 'Teshuvah'.
- Body font: 'PT Sans' a humanist sans-serif that combines a modern look and a little warmth; suitable for all body text and longer text blocks.
- Use simple, flat icons relevant to event categories and general rules.
- Employ a clear vertical rhythm with generous white space between sections.
- Ensure responsive grid layouts for events and schedule, adapting to various screen sizes.
- Incorporate subtle hover effects on buttons and cards for enhanced interactivity.
